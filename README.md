# PROG2_Arbeitsblaetter

## Informationen zum Projekt

Projekttitel:	Manchester United Informationen
Autor:			Athavan Sanganathapillai
Datum:			17.02.1994
Version:		v1.0

## Beschreibung

Dieses Projekt dient als eine Übung für den Unterricht im PRG2.
Das Ziel dieses Projektes ist eine Sammlung von Informationen über die Spieler des grössten Klubs der Welt Manchester United zu sammeln.
Der Ablauf wird sich im laufe des Projektes herauskristalisieren.
